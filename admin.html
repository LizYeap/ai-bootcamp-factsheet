📖 Plain-English Formatting Guide

Some fields support a small set of HTML tags to format text. Here's what each tag does, with before/after examples.

<br>
Line break — forces the text onto a new line. Like pressing Enter.
You type
39-07, Menara EcoWorld @ BBCC,<br>2, Jln Hang Tuah,<br>55100 Kuala Lumpur
Result on page
39-07, Menara EcoWorld @ BBCC,
2, Jln Hang Tuah,
55100 Kuala Lumpur
<strong>…</strong>
Bold text — makes words stand out visually. Use for important details like times, floor numbers, or names.
You type
Head to <strong>Level 39, Unit 07</strong>
Result on page
Head to Level 39, Unit 07
<em>…</em>
Italic text — a softer emphasis. Good for optional notes or side remarks.
You type
Headphones <em>(optional — some demos involve audio)</em>
Result on page
Headphones (optional — some demos involve audio)
Combining tags
You can mix tags — bold + italic, or bold + line break — by nesting them.
You type
Registration: <strong>9:30 AM</strong><br>Event: <strong>10:00 AM – 6:00 PM</strong>
Result on page
Registration: 9:30 AM
Event: 10:00 AM – 6:00 PM
⚠️ Fields that support HTML
  • Hero Tagline
  • Venue Address
  • Lobby Instructions
  • Parking instructions (per item)
  • Homework item descriptions
  • Items to Bring text
  • FAQ answers
